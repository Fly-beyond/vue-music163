# vue-music163

> A Vue.js project for music
### Use
* vuejs-2.0
* vue-router
* mint-ui
* vue-cli

## Support
* Chrome
* Firefox
* Mobile browser

## Package for app
* Appcan
* Apicloud


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
